什麼是軟文-ta的前世今生-ta的七十二變

發佈於 2021-06-26

​2013年以來，中國「 移動網際網路 」呈現爆炸式發展，隨著獲取信息變得越來越方便，傳統廣告服務越來越不被接受，軟性宣傳應聲而出。

2012年微信發布第一版公眾平臺，2014年新浪微博logo改名為「 微博 」。公眾自媒體成為了自由職業者們側重的方向，結合當下的移動社交平臺，越來越多的機構和職業人開始打造自己的自媒體，相對的文案創作需求量，越來越大，自由撰稿人也成了高薪，自由，文藝的代表職業。

但是很多人只是在消費這些文案，而不懂得如何編寫。不但要有流利的口才，熟練應用文字的能力，也應該是當下職業人和即將步入社會的大學生，必備的職場能力。

經常有人會問，什麼是軟文？

從概念上來解釋，軟文是一般含有軟性宣傳的文字內容。這樣的解釋只會讓人糊塗，我的解釋，就是除了原創小說的一切文字內容。軟文的「 軟 」正式它的精妙所在。

網際網路發展到現在已經到了人們不可預測的速度，但是信息傳播的方式，依舊是圖片和文字為主，其中文字，又是人們自古以來，接收信息的最信賴的方式。所以在有文字的地方就有「 軟性宣傳 」。

軟文的表現能力，不像硬性廣告那樣，讓人無法接受，甚至直接忽視。它是以一種用戶能接受的口吻傳遞信息，並且強化用戶腦中對產品的印象，達到宣傳效果，這才是軟文的妙處。

// 你身邊的軟文，你發現了嗎? //

軟文發展到現在已經不止是為了讓客戶當下就訂購產品，有時可能是為了加強產品在用戶腦中的印象，等客戶需要的時候，加強用戶選擇的傾向。這些情況會出現在很多人們容易忽略、放鬆警惕的地方。

比如這篇：

荊棘最開始沒看出有哪裡是營銷，並且買了文中推薦的四本書，受益匪淺，但是後來經網友指點才發現，其中三本書都是一家出版社出版的，這才意識到作者的高明。分享經驗的同時，推銷了產品。

再比如這篇：

百度百科每天都會有成千上百萬次閱讀量，裡面的詞條，只要用戶符合條件，就有權限編輯，這其中是否有軟文？
答案是肯定的。
就像是這篇文章，「輕食主義」詞條解釋，由來→延伸→攻略→提供對應書籍。人們會很習慣這樣的規律，從提出問題，到解釋問題，到延伸幫助。這樣的規律比一張精美的讀書海報，要更容易讓人接受，並且不會讓用戶產生任何反感，不覺得這是廣告，反而覺得這是幫助。
這樣的「陷阱」你發現了嗎？
// 軟文能幹什麼? //

很多人對軟文的第一印象就是「 廣告 」，賣產品。這的確是軟文的一個很大的作用。但軟文最大的作用是深入人心的影響力和病毒式的傳播力。

羅永浩、papi醬、張雪峰，這些網絡紅人，全都是靠著優秀的文案，製造了很多病毒話題，擴大了自己的影響力，從而帶起了自己的產品。軟文是帶有軟性營銷的文字，但是文字的使用方式就有很多變化了，所以不要自己局限軟文的使用方式。

可能你不需要做自媒體，可能你正在辦公室悠閒的消磨，可能你是自由職業者。但是軟文對你來說一樣有很多需要學習的理由。

想想你的周報，月報，在想想你的項目計劃書，想想明天的會議大綱，想想下一篇演講稿，甚至是你的簡歷。這些雖然不是為了推銷產品，但是他們都是要展示給其他人的，並且需要獲得認同。你需要做的就是在文章中，把自己的想法、業績、能力，當做產品，推銷給你的老闆、客戶、上司、面試官。

軟文和SEO也是綁在一起的。SEO中的黑帽、白帽，都是從本身或者採用作弊的方式提高傳播力，但這些都已經被用戶了解了，並且容易引起方案。移動網際網路發展到現在，人們的注意力，被無限制的分散，所以廣告，已經默認被用戶忽略。

人們更多的是在自己想要了解的信息中投入更多的，注意力。在用戶活躍度高的地方投放精心編排的軟文，把企業產品，藏在文中。好的軟文可以提高產品在用戶腦中的印象，進而擴大企業品牌的知名度和影響力。

// 軟文還有哪些你不知道的 //

被譽為新四大發明之一的「 網購 」，已經給人們帶來了很多便利。人們可以在彈指間找到自己想要的產品，還能貨比N家，查看使用報告，用戶測評，顧客售後評價，簡直不要太方便。現在，一個手機，已經可以解決人們絕大部分日常需求。

但是你想過這些，都是軟文嗎？

這可能讓很多人覺得不能接受，但是事實就是這樣的，有時候一個網站，一個品牌的搭建，初期是需要很多模擬數據來鋪墊的。

產品使用測評？

用戶使用反饋？

品牌可信度和影響力？

這些都有可能。網際網路時代人們需要了解的信息越來越多，對於剛剛出現的產品，人們總是抱著懷疑的態度。並且網際網路最大的弊端，仍然沒有實體行業的體驗感。所以人們需要很多其他數據來幫助自己做判斷。

雖然這是很多從業人員不承認的，這也可以理解，創業者為了跟上現在信息傳播的速度，也只能這樣做。

雖然軟文存在引導和欺瞞，但是它的出現加快了品牌的發展速度，可以為企業提供更多的關注度。況且軟文的拓展潛力是無限的，並且網際網路時代，信息傳遞的方式越來越多，也越來越快，軟文的發展方向也會逐漸偏向於「潛藏式廣告」。在無形中向用戶傳遞產品信息，這種軟性營銷手段，會是廣告業的未來方向，也是任何想要達到推銷效果的必選方式。
